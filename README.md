# Curriculum Vitae

## Installation & Usage

To run this project locally:

1. Clone or download this repo:
   ```bash
   git@github.com:KhatunaKhatuna/curriculum-vitae.git
   ```
2. Open the index.html file in any modern browser.

Or use github live page link

```bash
  https://khatunakhatuna.github.io/curriculum-vitae/
```

---

## Technologies Used

1. **HTML5:** Semantic structure for content
2. **CSS3:** Flexbox and grid layout, transitions, media queries for responsiveness
3. **JavaScript:** Event listener, setInterval and class toggling to manage interaction
